# 🤖 Бот для оценки преподавателей Московского политеха

Telegram-бот для сбора обратной связи о преподавателях с возможностью оценки по критериям, выбором тегов и системой модерации.

## 📋 Функциональность

### Для пользователей:
- Оценка преподавателей по вопросам (от 1 до 5)
- Выбор тегов (3-10 шт.)
- Комментарий к отзыву (опционально)
- Просмотр рейтингов и отзывов других пользователей
- Жалобы на некорректные отзывы

### Для администраторов:
- Управление преподавателями (CRUD)
- Модерация жалоб
- Управление тегами и вопросами опросника
- Черный список слов для фильтрации комментариев
- Экспорт данных в CSV
- Статистика и отчеты

## 🛠️ Установка и запуск

### 1. Установите Python

Скачайте с официального сайта: https://www.python.org/downloads/

**Важно:** При установке обязательно поставьте галочку **«Add Python to PATH»**.

### 2. Скачайте папку проекта

### 3. Создайте виртуальное окружение

Откройте терминал в папке проекта и выполните:

```bash
python3 -m venv venv
```

### 4. Активируйте виртуальное окружение

**Windows (CMD или PowerShell):**
```bash
venv\Scripts\activate
```

**macOS / Linux:**
```bash
source venv/bin/activate
```

После активации в начале строки терминала появится `(venv)`.

### 5. Установите библиотеки

```bash
pip install -r requirements.txt
```

### 6. Получите токен у BotFather

- Откройте Telegram и найдите бота **@BotFather**
- Если создаете нового бота: `/newbot` → следуйте инструкциям
- Если бот уже существует: `/mybots` → выберите бота → **API Token**
- Скопируйте строку токена

### 7. Добавьте токен в проект

Создайте в папке проекта файл **`.env`** и запишите:

```
BOT_TOKEN=ваш_токен_здесь
```

### 8. Настройте администраторов

В файле `handlers/admin_handlers.py` найдите строку:

```python
ADMIN_IDS = []
```

Замените ID на ID ваших Telegram-аккаунтов (можно узнать у бота @userinfobot).

### 9. Запустите бота

```bash
python3 app.py
```
