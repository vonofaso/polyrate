# Инструкция для запуска бота

### 1. Подготовка окружения
**Установите Python**
Скачайте с официального сайта: https://www.python.org/downloads/
**Важно:** При установке обязательно поставьте галочку **«Add Python to PATH»**.

### 2. Скачайте папку проекта

### 3. Создайте виртуальное окружение
Откройте в папке проекта и выполните:
```bash
python3 -m venv venv
```

### 4. Активируйте виртуальное окружение
Windows (CMD или PowerShell):
```bash
venv\Scripts\activate
```
macOS / Linux:
```bash
source venv/bin/activate
```
После активации в начале строки терминала появится (venv).

### 5. Установите библиотеки
```bash
pip install -r requirements.txt
```

### 6. Получите токен у BotFather
Откройте Telegram и найдите бота @BotFather.
Если вы создаете нового бота, напишите /newbot и следуйте инструкциям.
Если бот уже существует, напишите /mybots, выберите своего бота -> API Token.
Скопируйте строку токена (выглядит так: 123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11).

### 7. Перенесите токен в проект
Создайте в папке проекта файл .env и запишите туда:
BOT_TOKEN=123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11

### 8. Запуск бота
```bash
python3 app.py
```
